## 手机进销--管理系统



### 项目介绍



#### 该系统前后端分离，api使用restful协议，后端使用Python，Django，DRF等技术，前端代码使用element进行构建，现金管理销售管理，库存管理等业务管理流程

###  硬件要求及开发环境

- Python版本为V3.9+
- Django版本为V3.2+
- Django-rest-framework版本为V3.12+
- Vue版本为2.6+
- 数据库为MySQL
- 前端组件为Elenment
